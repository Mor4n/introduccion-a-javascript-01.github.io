# Módulo 1 - Introducción a JavaScript

Este repositorio contiene las actividades y proyectos realizados durante el Módulo 1 de Dev.F: **"Introducción a JavaScript"**.

## Estructura del repositorio

- **01-introducción-a-javascript/**  
  Tarea sobre los tipos de datos en JavaScript

- **02-comparaciones-y-decisiones/**  
  Tarea sobre los comparadores y tomas de decisión en JavaScript

## ¿Cómo usar este repositorio?

1. Navegue a la carpeta del proyecto que te interese.
2. Lea el archivo README.md de cada proyecto para más detalles.
3. Abre los archivos .html y/o .js en tu navegador para ver los ejemplos en acción o accede a la página de Github pages especificada en cada README para ver los archivos
